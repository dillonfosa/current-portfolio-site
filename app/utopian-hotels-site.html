A luxury hotel site design created using the following.
HTML5 check
CSS3 check
JQUERY check
PHP check

FULLY RESPONSIVE


The idea behind the design

This design was to showcase a high-end luxury hotel company. My concept was to implement the use of simple material cards while making the images pop. The site is fully responsive and works smoothly with jQuery as well as CSS3 transitions.

Mobile First Approach

The key was designing this from the traveling user's perspective. This means start mobile design first. Designing for those running to catch a late flight. The site is clean and simple to navigate.
